# 🚀 AuthNexus
### Scalable JWT Authentication System with Redis & Role-Based Access Control

AuthNexus is a production-style authentication system built using **Spring Boot**, **JWT**, **Redis**, and **RBAC**.  
It demonstrates secure session management using access and refresh tokens, token validation, and frontend integration with Axios interceptors.

---

## 📌 Features

- 🔐 JWT-based stateless authentication
- 🔁 Refresh token flow with Redis storage
- ⚡ Scalable session management
- 🛂 Role-Based Access Control (RBAC)
- 🌐 React + Axios interceptor integration
- 🔒 Secure logout & token invalidation
- 🧱 Clean layered architecture
